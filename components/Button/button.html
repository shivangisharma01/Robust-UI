Button
A button is a widget used to perform actions such as submitting a form, opening a dialog, canceling an action, or performing a command such as inserting a new record or displaying information.
Solid Button
Solid button is simple basic textual button used for primary functions, which mostly is based on website theme color.
Icon Button
Button with icons are generally used to highlight the role performed and to seek the user attention towards it.
Float Action Button
These buttons performs most common action on screen and are fixed to a particular part of the viewport and are stacked on top of all the things.
Button with link
Button with links and basically used when you have to redirect the user onto the another website or on diffrent segment on the same webpage.
