Lists
Lists are used to specify lists of information. All lists may contain one or more list elements. There are four different types of HTML lists: Ordered List,Unordered List,Description List and Nested List.
Ordered List
In the ordered lists all the list items are marked with numbers by default. It is known as numbered list also. The ordered list starts with ol tag.
  1. Car
  2. Sports Bike
  3. Rocket
Unordered List
In Unordered list all the list items are marked with bullets, you can also change the bullets style with the help of type attribute. It is also known as bulleted list also. The Unordered list starts with ul tag.
  • Car
  • Sports Bike
  • Rocket
Description List
Description list it is also known as definition list where entries are listed like a dictionary or encyclopedia. Description list contains following three tags: 1 dl tag- defines the start of the list. 2 dt tag- defines a term. 3 dd tag- defines the term definition (description).
Aries
-Birthday month March 21 - April 20.
Pisces
-Birthday month February 19 - March 20.
Sagittarius
-Birthday month November 23 - December 21.
Nested List
A list within another list is termed as nested list. You can add any list in one-another, here is the example of unordered list within the ordered list.
  1. Shopping list
    • Mlik
    • Cheese
    • Chocolate
  2. Horoscope Names from February-May
    • Aquarius
    • Pisces
    • Aries
    • Gemini
